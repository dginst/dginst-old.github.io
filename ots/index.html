OPENTIMESTAMPS

A STEP BY STEP TUTORIAL

A timestamp proves that some data existed prior to some point in time. OpenTimestamps (OTS) defines a set of operations for creating provable timestamps and later independently verifying them.

Github project: https://github.com/lvaccaro/javascript-opentimestamps-tools

HASH DATA FILE


Get the hash value associated to the data file to be stamped.
No internet connectivity is required: the hash is calculated inside your browser preserving your privacy.


TIMESTAMP HASH VALUE


Submit the hash value to the OpenTimestamps calendar servers for attestation of its existance (timestamp) and save the OTS proof as local file; internet connectivity is required.
Anyone could realize a timestamp with the permissionless blockchain(s) by paying the transaction fees; conveniently, OpenTimestamps performs this operation for free without any registration or api key.



LOAD OTS PROOF FROM FILE


Load the OTS proof from a local file (no internet connectivity required).


DISPLAY OTS PROOF INFO


Display the informations included in the OTS proof (no internet connectivity required).


UPGRADE OTS PROOF


Upgrade the OTS proof with the attestations obtained by the OpenTimestamps calendars (internet connectivity is required).


VERIFY OTS PROOF


Verify the attestations included in the OTS proof (internet connectivity is required).
Proof upgrading is attempted before verification: if an upgrade is available, then it is downloaded and used for verification.
